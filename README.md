\## SDG Alignment

SDG 3 (Health), SDG 4 (Education), SDG 10 (Reduced Inequalities)



\## Tech Stack

\- Streamlit (UI)

\- YOLOv8 (Object Detection)

\- EasyOCR (Text Recognition)

\- Google Gemini 2.5 Flash (Scene Description)

\- gTTS (Text-to-Speech)

\- Docker (Containerization)



\## Quick Start



\### Using Docker

docker pull adityamenon17/visionvoice-pro:latest

docker run -p 8501:8501 adityamenon17/visionvoice-pro:latest



\### Local Development

pip install -r requirements.txt

streamlit run app.py



Open http://localhost:8501
